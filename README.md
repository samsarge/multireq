# Multireq
For documentation
`cargo run -- --help`

Send multiple requests to an endpoint

Usage: `multireq [OPTIONS] --url <URL> --method <METHOD>`
```
Options:
  -u, --url <URL>
  -m, --method <METHOD>  [possible values: get, post, put, patch, delete]
  -x, --amount <AMOUNT>  [default: 1]
  -h, --help             Print help
  -V, --version          Print version

```
